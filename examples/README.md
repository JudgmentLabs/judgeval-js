# Judgeval JS Examples

Collection of example projects demonstrating various features of the Judgment SDK.

## Setup

Before running examples, set your environment variables:

```bash
export JUDGMENT_API_KEY="your-api-key"
export JUDGMENT_ORG_ID="your-org-id"
export JUDGMENT_API_URL="https://api.judgmentlabs.ai"
export OPENAI_API_KEY="your-openai-api-key"
```

## Running Examples

Use the convenience script from the project root:

```bash
bun run example <example-name>
```

Or run directly:

```bash
cd examples/<example-name>
npx tsx index.ts
```
