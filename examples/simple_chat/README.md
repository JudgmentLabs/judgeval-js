# Simple Chat Example

Demonstrates basic tracing of a chat application using OpenTelemetry OpenAI instrumentation.

## Setup

Set environment variables:

```bash
export JUDGMENT_API_KEY="your-api-key"
export JUDGMENT_ORG_ID="your-org-id"
export OPENAI_API_KEY="your-openai-api-key"
```

## Run

```bash
npm run example simple_chat
```

Or directly:

```bash
cd examples/simple_chat
npx tsx index.ts
```
